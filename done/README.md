# done

Usage:

```bash
done is a CLI Task Manager that helps you to manage your tasks - Yup a TODO application

Usage:
  done [command]

Available Commands:
  add         Adding a task to a task manager
  clean       Cleaning the task list.
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  list        List all the tasks in the manager
  scratch     Scratch a task as done.

Flags:
  -h, --help   help for done

Use "done [command] --help" for more information about a command.

```
